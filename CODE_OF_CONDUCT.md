# 行動規範

## 基本姿勢

互いに敬意を払い、具体的かつ建設的に対話してください。経験、属性、言語、背景の異なる参加者を歓迎します。コードや提案への批判は内容に向け、個人への攻撃にしないでください。

嫌がらせ、脅迫、差別、私的情報の公開、性的またはその他の望まれない言動は認めません。

## 適用範囲

この規範は、Issue、Pull Request、Discussion、security report、および本プロジェクトを代表して参加するその他の場所に適用されます。

Maintainer は、規範に反する投稿や変更を編集、非表示、削除でき、状況に応じて参加を一時的または恒久的に制限できます。

## 報告

通常の行動上の問題は、適切な GitHub の非公開連絡手段から Maintainer へ報告してください。Security に関係する内容は、[SECURITY.md](SECURITY.md) に記載した private vulnerability reporting を使用してください。

第三者の私的情報を公開 Issue に投稿しないでください。Maintainer は報告を誠実に確認し、公開範囲を必要最小限に抑え、状況・影響・反復性に応じた対応を行います。
